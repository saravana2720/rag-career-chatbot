# ATS Keywords for Technical Resumes

## What is ATS?
ATS (Applicant Tracking System) scans resumes for keywords before a recruiter sees them.

---

## Common ATS Keywords (Software Roles)

### Programming Languages
Python, Java, JavaScript, C++, SQL

### Frameworks & Libraries
React, Angular, Node.js, Flask, Django

### Databases
MySQL, PostgreSQL, MongoDB, Oracle

### Cloud & DevOps
AWS, Docker, Kubernetes, CI/CD

### Data & AI
Machine Learning, Deep Learning, TensorFlow, Pandas, NumPy

---

## Action Verbs (Very Important)
- Developed
- Implemented
- Designed
- Optimized
- Automated
- Analyzed
- Deployed

---

## Keywords by Experience Level

### Fresher
Internship, Academic Project, Certification, Training

### Experienced
Scalable Systems, Performance Optimization, Microservices

---

## ATS Optimization Tips
- Match keywords from job description
- Avoid tables and images
- Use standard headings (Skills, Experience, Education)
- Do not use fancy fonts or icons

---

## Example ATS-Friendly Skill Line
Developed REST APIs using Python and Flask, integrated with MySQL database, and deployed on AWS.
