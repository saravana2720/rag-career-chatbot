# Resume Tips for Freshers

## Objective Section
- Keep it short (2â€“3 lines)
- Mention your role and key skills
- Avoid generic statements

**Example:**
Motivated Computer Science graduate with strong fundamentals in Python and Machine Learning, seeking an entry-level software role.

---

## Education
- Mention degree, college name, and year
- Add CGPA if above 7.0
- Include relevant coursework

---

## Skills Section
- Separate technical and soft skills
- Avoid exaggeration
- Mention tools you have actually used

**Example:**
- Programming: Python, Java
- Web: HTML, CSS, React
- Tools: Git, Docker

---

## Projects
- Very important for freshers
- Explain problem, solution, and tech used
- Add GitHub or demo link

**Format:**
- Project Name  
- Problem Statement  
- Technologies Used  
- Outcome  

---

## Internships / Training
- Mention duration and company
- Explain what you learned
- Add measurable impact if possible

---

## Common Mistakes to Avoid
- More than 2 pages
- Spelling and grammar errors
- Fake experience
- Unnecessary personal details

---

## Final Tips
- Keep resume to 1 page
- Use simple fonts
- Save as PDF
